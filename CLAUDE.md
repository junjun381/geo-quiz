# geo-quiz

世界地図クイズアプリ

## 概要
地域別に国の場所を覚えるためのクイズアプリ

## 技術スタック
- React 18 + TypeScript
- Vite
- Natural Earth（地図データ、パブリックドメイン）

## 機能
- 世界地図から地域を選択してズーム
- 3つのクイズモード（穴埋め、国名→地図、地図→国名）
- 多言語対応（日本語、英語、中国語、韓国語、スペイン語）
- 1言語 or 2言語併記の切り替え
- 進捗保存（localStorage）

## 地域
- 中東、アフリカ、ヨーロッパ、アジア、オセアニア、北米、南米
